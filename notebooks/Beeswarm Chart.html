Beeswarm Chart¶

Source: Better Data Visualization Using Beeswarm Chart

  • One-dimensional chart (see also Box plot and a Violin plot)
  • Up to 2 variables:
    • X axis (quantitative variable)
    • Dot color (qualitative variable)
  • It emphasizes individual points in a distribution instead of binning them like a histogram.
  • Limitations:
    • data size capacity: too many same values
      • solution: change scale (linear and logarithmic)
